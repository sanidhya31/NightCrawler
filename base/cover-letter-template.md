# Cover Letter Template

The skill generates `cover-letter.md` (EN) and `cover-letter.de.md` per job using
this structure. Keep it simple: letterhead + body + regards. The letterhead (name +
contact) is pulled automatically from `base-resume.json` — do not hardcode it.

## Structure

```
<Name>                          <- from base-resume.json candidate.name
<Location · Phone · Email · LinkedIn · GitHub>   <- from base-resume.json

<Date>

<Company name>
Re: Application for <Role title>

Dear Hiring Team,

<Paragraph 1 — who I am + why this role/company. 2-3 sentences.>

<Paragraph 2 — 2-3 concrete, true reasons I fit, drawn from the tailored resume
(lead with the strengths the job wants; tech as support if relevant). Honest:
never claim skills/levels the base resume does not support.>

<Paragraph 3 — short close: availability, enthusiasm, invite to talk.>

Kind regards,
<Name>
```

## Rules
- Letterhead = name + contact line only (no fake company address block).
- Tone: professional, clear, slightly warm. No buzzword salad.
- ~250-300 words, one page.
- Same honesty rules as the resume (no invented facts; respect real language levels).
- DE version: natural professional German; keep contact details and proper nouns as-is.
- Small hyphens "-", no em-dashes.
